HTML Рђћ рЃЏрЃЮрЃЎрЃџрЃћрЃЊ

HTML рЃљрЃарЃўрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ. рЃЏрЃўрЃАрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃбрЃћрЃЦрЃАрЃбрЃў, рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў, рЃџрЃўрЃюрЃЎрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў. рЃАрЃўрЃбрЃДрЃЋрЃљ HTML рЃюрЃўрЃерЃюрЃљрЃЋрЃА HyperText Markup Language.

<p>рЃћрЃА рЃљрЃарЃўрЃА рЃЕрЃћрЃЏрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃў.</p>
